# 🚀 Personal Finance Budget Tracker with AI Advisor

## 📌 Project Overview

Personal Finance Budget Tracker is a modern AI-powered financial management application designed to help users track income, expenses, budgets, savings, and financial habits in real time.

Unlike traditional expense trackers that only store transactions, this system analyzes financial data, predicts future expenses, provides intelligent financial advice, generates spending alerts, and allows users to interact with an AI Financial Assistant.

The application combines Mobile Development, Backend APIs, Database Management, Artificial Intelligence, OCR Technology, and Voice Recognition into a single platform.

---

# 🎯 Problem Statement

Most existing finance applications only provide:

* Income tracking
* Expense tracking
* Budget recording
* Static reports

These systems do not help users understand:

* Future spending patterns
* Savings opportunities
* Budget risks
* Financial decision making

Users are forced to manually analyze their financial data.

This project solves this problem by introducing AI-driven financial insights and automation.

## 🎤 Voice Expense Entry

Users can enter expenses using voice commands.

Examples:

* "Five hundred"
* "Food"
* "UPI"
* "15 slash 06 slash 2026"

The system automatically converts speech into structured expense records.

## 📷 Smart Receipt Scanner

Using OCR (Optical Character Recognition), users can scan bills and receipts.

The system automatically detects:

* Amount
* Category
* Payment Method
* Date

and creates expense entries automatically.
